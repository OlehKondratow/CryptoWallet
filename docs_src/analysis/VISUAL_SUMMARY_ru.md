## 🔄 Диаграммы Потока Данных

### stm32_secure_boot - Поток HID Подписанта

```
Ввод Пользователя (Кнопка/UART/HID)
         ↓
    ┌────────────────────┐
    │ Транспорт подписи   │
    │ - Разобрать команду │
    │ - Валидировать вход │
    └────────────────────┘
         ↓
    ┌────────────────────┐
    │ Управление ключами  │
    │ - Загрузить ключ    │
    │ - Только в памяти   │
    └────────────────────┘
         ↓
    ┌────────────────────┐
    │ Ядро подписания    │
    │ - Хеш SHA-256      │
    │ - Подпись ECDSA    │
    │ - Библиотека CMOX  │
    └────────────────────┘
         ↓
    ┌────────────────────┐
    │ Ответ             │
    │ - UART или HID     │
    │ - Подпись         │
    └────────────────────┘
```

### CryptoWallet - Полный Поток Кошелька

```
Сетевой Ввод (HTTP/WebUSB/UART)
         ↓
    ┌─────────────────────────┐
    │ Обработка Протокола     │
    │ - Разобрать JSON/двоично│
    │ - Несколько транспортов │
    └─────────────────────────┘
         ↓
    ┌─────────────────────────┐
    │ Валидация Транзакции    │
    │ - Адрес (Base58/Bech32) │
    │ - Проверка суммы        │
    │ - Поддержка валют       │
    └─────────────────────────┘
         ↓
    ┌─────────────────────────┐
    │ Развертывание Ключа     │
    │ - Путь HD BIP-32        │
    │ - Библиотека trezor     │
    │ - Генерация дочер. ключ │
    └─────────────────────────┘
         ↓
    ┌─────────────────────────┐
    │ Подтверждение Пользова  │
    │ - Требуется нажатие     │
    │ - Информация OLED       │
    │ - Подтвердить/Отклонить│
    └─────────────────────────┘
         ↓
    ┌─────────────────────────┐
    │ Ядро Подписания         │
    │ - SHA-256 + HMAC-SHA512 │
    │ - ECDSA secp256k1       │
    │ - Nonce RFC 6979        │
    │ - Кодирование DER       │
    └─────────────────────────┘
         ↓
    ┌─────────────────────────┐
    │ Очистка Памяти          │
    │ - Буферы memzero()      │
    │ - Безопасная очистка    │
    │ - Предотвращение утечки │
    └─────────────────────────┘
         ↓
    ┌─────────────────────────┐
    │ Ответ                  │
    │ - Подпись + метаданные │
    │ - Вывод мультипротокол │
    └─────────────────────────┘
```
