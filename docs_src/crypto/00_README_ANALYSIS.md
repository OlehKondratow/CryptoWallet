## 🔍 Ключевые находки

### Архитектура trezor-crypto

```
┌─────────────────────────────────────────┐
│ Приложение (task_sign, task_net)       │
├─────────────────────────────────────────┤
│ Адаптер: crypto_wallet.c/h             │  ← 6 публичных API функций
├─────────────────────────────────────────┤
│ trezor-crypto:                          │
│ • BIP-39 (12 слов из 128 бит)          │
│ • BIP-32 (отведение m/44'/0'/0'/0/0)   │
│ • ECDSA (secp256k1 подписание)         │
│ • SHA-256 (хеширование)                │
│ • Base58Check (валидация адреса)       │
├─────────────────────────────────────────┤
│ STM32H7 HAL (TRNG, GPIO, UART)         │
└─────────────────────────────────────────┘
```

### RNG механизм

```
TRNG (32 бит)  ╲
              XOR → LCG Update → Выход
LCG Pool       ╱
```

- **TRNG:** STM32H743 hardware RNG (аппаратный)
- **LCG:** Numerical Recipes параметры (1664525, 1013904223)
- **Комбинация:** XOR для защиты от отказа одного источника

### Dieharder тестирование

```
Файл: 128 MiB сырых RNG байт
Тесты: ~100+ встроенных тестов
Время: 30 мин — 3 часа
Критерий PASS: p-value 0.001 - 0.999
```
