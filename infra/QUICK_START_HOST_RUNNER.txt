# Host Runner - Quick Start

## 🚀 30-Second Start

### 1. Run setup script

```bash
cd /data/projects/CryptoWallet/infra
chmod +x setup-host-runner.sh
./setup-host-runner.sh
```

Script will ask for a token. **Where to get it:**

1. Open: http://localhost:3000/admin/runners
2. Click "Create new Runner" (blue button)
3. Copy token
4. Paste into script

### 2. Check status

```bash
systemctl --user status gitea-runner
```

Expected: **Active (running)**

### 3. Check in Gitea UI

http://localhost:3000/admin/runners

You should see 2 runners:
- Container (ID 6) - ACTIVE ✅
- Host runner - ACTIVE ✅ (new)

---

## 🧹 Clean Up Old Runners

In Gitea UI:
1. **Admin → Runners Management**
2. For each OFFLINE runner (ID 3, 4, 5):
   - Click **Edit**
   - Click **Delete**

---

## 🧪 Testing

### Run CI (simple-ci):

```bash
cd /data/projects/CryptoWallet
git add .gitea/workflows/simple-ci.yml
git commit -m "ci: trigger Simple CI"   # if file already exists in repo, empty commit + push is enough
git push
```

Then in Gitea: **Actions** -> **Simple CI** -> if needed, click **Run workflow** (when `workflow_dispatch` exists).
